Frequently Asked Questions

Common questions about our document verification service

What file formats does DocVerify support?

DocVerify supports common document formats including PDF, JPEG, PNG, and TIFF files. We recommend using high-quality scans or images for the best verification and OCR results.

How accurate is the document forgery detection?

Our machine learning models have been trained on extensive datasets and can detect various types of document manipulation with high accuracy. However, results may vary depending on document quality and type of alterations.

What languages does the OCR support?

Our OCR technology supports multiple languages including English, Spanish, French, German, and many others. The system automatically detects the document language and extracts text accordingly.

Is my document data secure?

Yes, we take data security seriously. All documents are processed using encrypted connections (HTTPS), and files are automatically deleted from our servers after processing is complete.

Can I process multiple documents at once?

Yes, DocVerify supports batch processing. You can upload multiple documents simultaneously, and our system will process them efficiently and provide individual reports for each document.

How long does document processing take?

Most documents are processed within seconds. Processing time may vary depending on document size, complexity, and current system load. You'll receive real-time updates during the analysis process.

Do you provide an API for integration?

Yes, we offer a RESTful API that allows you to integrate DocVerify's document verification and OCR capabilities into your existing applications and workflows. Contact us for API documentation and access.
